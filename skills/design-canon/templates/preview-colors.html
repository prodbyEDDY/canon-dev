Semantic colors

Только для семантики, не для категоризации

Success — --success
Positive deltas (+12%), success-toasts. Никогда CTA.
Warning — --warning
Approaching due dates, partial-data states.
Danger — --danger
Errors, destructive actions, overdue.
Info — --info = brand-500
Informational toasts, focus rings, chart accent.

Text colors

Primary text — fg-1

Заголовки, KPI-числа, основной текст в таблицах. Same as brand-900.

color: var(--fg-1);

Body text — fg-2

Основная масса параграфов, описания, контекст под заголовками.

color: var(--fg-2);

Caption — fg-3

Подписи под полями, meta-данные, hints, eyebrow-labels.

color: var(--fg-3);

Disabled — fg-4

Placeholders в input'ах, disabled-состояния.

color: var(--fg-4);
