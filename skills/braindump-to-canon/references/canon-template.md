> Этот файл — заготовка для генерации канона любого модуля. Подставь имя модуля и заполни секции. Не все секции обязательны для маленьких модулей — но **порядок сохраняй**.

```markdown
Разрешённые переходы:
```
draft → active → closed
draft → cancelled

```
Запрещены:
- Прямой переход `draft → closed` без `active`
```

## Заметки для генерации

- Заполняй каждую секцию **конкретно**, не плейсхолдерами. Если секция не применима для конкретного модуля — удали её, **не оставляй "TODO"**.
- Длина канона зависит от сложности модуля. Простой модуль — 2-4 страницы. Сложный (как Финансы) — 20-40 страниц.
- Цитаты из законодательства / стандартов — приветствуются, особенно для регулируемых доменов.
- Используй em-dash для связок, «ёлочки» для названий, sentence case для заголовков. См. `docs/BRAND-VOICE.md`.
